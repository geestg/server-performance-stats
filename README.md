# Server Performance Stats

Skrip ini digunakan untuk menampilkan metrik performa dasar pada sistem Linux melalui Bash.

## Fitur
- Penggunaan CPU
- Penggunaan memori (free vs used + persentase)
- Penggunaan disk (free vs used + persentase)
- Top 5 proses berdasarkan penggunaan CPU
- Top 5 proses berdasarkan penggunaan memori
